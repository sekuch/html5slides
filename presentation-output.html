Motivation

  • Schnelleres erstellen von Präsentationen
  • Kein Layouten
  • Keine umständlicher Menüführung
  • Einbinden von Dynamischen Inhalten
  • Automatisches Syntax-Highlighting

Technik

Erstellen der Präsentation

  • Layout in CSS
  • Schreiben in Markdown
  • Markdown -> HTML
  • Präsentieren

Markdown

  • Strukturierte Sprache
  • Leicht in HTML umzuwandeln
  • Bekannt duch Einsatz in Wikis und Foren

Markdown

    # Titel

    Ein wenig Text

    - Item 1
    - Item 2

    ![alt text](/path/to/image.png "Title")

Markdown -> HTML


<h1 id="titel">Titel</h1>

<p>Ein wenig Text</p>

<ul class="build fade">
<li>Item 1</li>
<li>Item 2</li>
</ul>

<p><img src="/path/to/image.png" alt="alt text" title="Title" /></p>
